# Backend context

- The Python version of panoptikon (../panoptikon) is **deprecated**. All backend development happens only on the Rust port, and this UI targets the Rust gateway API.
- Any reference to a backend issue, bug, or possible improvement implicitly refers to the Rust port, unless a message explicitly says otherwise.

# Commit messages

- Keep commit messages short: a single subject line of at most 80 characters.
- Never add a `Co-Authored-By` line or any other trailer to commit messages.
